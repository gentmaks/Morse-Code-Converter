## Getting Started

Clone the repo

```bash
git clone https://github.com/gentmaks/Morse-Code-Converter.git
cd Morse-Code-Converter
```

If on windows:

- [Have vivado installed](https://www.xilinx.com/content/xilinx/en/support/download.html/)
- Open Vivado Tcl Shell
- Run the following commands


```bash
source build.tcl 
start_gui
```

running the setup script build.tcl generates the vivado project. Open the project 
using the GUI and take the project through the usual synthesis/implementation/bitfile generation flow
